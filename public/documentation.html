myFlix API Instruction's

The users will be able to use the myFlix application whenever they like to read information about different movies or update their user information.
The server-side of the myFlix application will ensure they have access to this information, that their requests can be processed, and that all necessary data can be stored.

REQUEST HTTP Method URL RESPONSE
Gets the list of data about ALL movies GET /movies Return a list of all movies
Gets the data about a single movie GET /movies/:title Return data about a movie by title
Gets the data about a Genre of movie GET /movies/genre/:genreName Return data about a genre by name
Gets the data about a Director of movie GET /movies/director/:directorName Return data about a director by name
adds new users POST /users Allow new users to register
update user info. PUT /users/:id Allow users to update their user info
Adds favoritemovie POST /users/:id/:movieTitle Allow users to add a movie to their favorites
remove a movie from favorites DELETE /users/:id/:movieTitle Allow users to remove a movie from their favorites
remove a user DELETE /users/:id Allow existing users to deregister
